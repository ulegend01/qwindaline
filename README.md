## ðŸ›  Deployment Guide (Vercel)

The easiest way to deploy is using **Vercel** (creators of Next.js).

1.  **Push to GitHub**:
    - Initialize git if not done: `git init && git add . && git commit -m "Initial commit"`
    - Create a new repo on GitHub and push.

2.  **Deploy on Vercel**:
    - Go to [Vercel.com](https://vercel.com) and sign up/login.
    - Click **"Add New Project"**.
    - Select your GitHub repository.
    - Click **"Deploy"**.
    - *Vercel will automatically detect Next.js and build your site.*
